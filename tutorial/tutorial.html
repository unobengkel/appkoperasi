Cara Setup dan Instalasi

Aplikasi Sistem Manajemen Koperasi ini dibangun menggunakan fondasi Web standard dipadukan dengan Python (FastAPI). Ikuti langkah-langkah di bawah ini untuk menyiapkan lingkungan pengembangan Anda.

Prasyarat Sistem

  • Python 3.8 atau yang lebih baru
  • Browser modern (Chrome/Edge/Firefox)

Langkah 1: Download/Membuka Proyek

Pastikan Anda sudah berada di folder proyek aplikasi koperasi melalui Command Prompt atau terminal text editor Anda (VSCode/Sublime).

Langkah 2: Menginstal Dependensi

Aplikasi web backend membutuhkan library FastAPI, Uvicorn, dan Pydantic. Anda dapat menginstalnya sekaligus melalui perintah:

pip install -r requirements.txt

Cara Menjalankan Server (Biasa)

Mode ini adalah cara standar untuk uji coba dan pengembangan secara instan.

1. Menjalankan Uvicorn

Buka terminal Anda di folder direktori proyek yang memiliki file main.py, lalu ketik:

uvicorn main:app --reload

Keterangan: Opsi --reload digunakan supaya saat ada perubahan pada program, server merestart secara otomatis.

2. Mengakses Aplikasi Web

Setelah pesan terminal mengatakan "Application startup complete", Anda bisa membuka rute server langsung di web browser:

  • http://127.0.0.1:8000 atau localhost:8000

Server akan secara otomatis me-redirect (mengarahkan) Anda ke halaman Input Data Anggota.

Cara Menjalankan Menggunakan Production/Process Manager

Jika Anda ingin menjalankannya di server VPS secara terus-menerus tanpa terhenti saat terminal di-close, disarankan menggunakan PM2.

Persiapan NPM & PM2

Anda membutuhkan Node.js. Setelah instalasi Node.js, jalankan perintah ini (sekali saja):

npm install -g pm2

Memulai via PM2

Di direktori proyek web Anda, jalankan perintah ini:

pm2 start "uvicorn main:app --port 8000" --name "koperasi-server"
pm2 start "python3 -m uvicorn main:app --host 0.0.0.0 --port 10011" --name "koperasi-server"
  • Cek log dengan status aplikasi: pm2 logs koperasi-server
  • Tutup atau hentikan dengan: pm2 stop koperasi-server

Cara Penggunaan Halaman Input Data Anggota

Halaman Input Data digunakan sebagai halaman pintu gerbang/penerimaan anggota per sesi.

1 Pengisian Biodata

Isikan form secara lengkap. Sistem akan menghitung dan mengisi Umur secara otomatis ketika tanggal lahir dipilih dari pop-up kalender.

2 Penyimpanan

Tekan tombol Simpan. Status akan tersemat "Success" secara otomatis di server database. Tabel sesaat di bagian bawah akan ter-update sesaat.

Fungsi Tabel Sementara

Tabel pada mode Input bertujuan hanya sebagai pencatatan atau riwayat pendaftaran bagi operator untuk sesi yang sedang berlangsung. Tabel ini akan hilang dari layar bila di-refresh, untuk kemudahan input baru di masa kerja berikutnya.

Cara Penggunaan Halaman Daftar Anggota

Halaman ini adalah *Master View* dari seluruh tabel database Koperasi yang sesungguhnya.

  • Pencarian Universal Asynchronous Gunakan kotak pencarian di bagian atas jika ingin mencari anggota dengan nama tertentu dan klik Cari. Proses ini mencari data melalui RAM sementara web.
  • Edit / Delete Data Universal Jika ada baris data yang cacat, klik/ketuk persis di barisan anggota tersebut. Akan muncul jendela Form Melayang (*Pop-Up Modal*). Di tempat tersebut, Anda bebas untuk mengedit semua bidang lalu klik Edit, atau klik Delete.
  • Cetak Data Menjadi Excel (.xlsx) Anda dapat mengunduh seluruh baris yang tersisa (terfilter) di dalam tabel ke file berwujud Data Excel.
